# Bézier Curve Visualizer

## About
Interactive Bézier curve visualizer written in C using raylib and raygui.  
Implements [De Casteljau's recursive algorithm](https://en.wikipedia.org/wiki/De_Casteljau%27s_algorithm) to display intermediate steps and polygonal chains.
